PIXEL POLICE

A design token integration testing tool

GitHub

Analysing a page to see if it's deviated from design tokens can be tedious and time consuming. It relys on having a an excellent eye for design and having the time to go through each page. As design tokens and websites evolve over time, without regular reviews there's a danger that design debt can increase and ruin visual consistency.

Pixel police automates this process and reports back when a design token hasn't been used.

  • If a villain has sneaked in a new background color, Pixel police will catch it.
  • If a bandit has missused a border, Pixel police will report it.
  • If a scoundrel has smuggled in a font, Pixel police will seize it.

Install

CLI

Option Type Description
-c, --config String Path to config file
-v, --verbose Boolean Output all test data (default: false)
-V, --version Output the version number
-h, --help Output usage information
