PIXEL POLICE

A design token integration testing tool

GitHub

Pixel Police makes it easy to check if pages on your website are using design tokens as you intended.

You tell it what your design tokens are and which css properties they can be used on, and Pixel Police will let you know what crimes have been committed

  • If a vandal defaces a page with the wrong font, Pixel police will find it
  • If a smuggler dashes accross a border, Pixel police will catch it
  • If a shady suspect uses an illegal color, Pixel police will seize it

Get started

Install

Create a config.js file and populate it with your design tokens

Add to your package.json

Run in terminal

CLI

Option Description
-c, --config Path to config file
-v, --verbose Output all test data (default: false)
-V, --version Output the version number
-h, --help Output usage information

Config

Attribute Type Description
urls Array[String] An array of urls that you would like to run the tests against
tests Array[Object] An array of test objects
tests.viewports.width Object/Number (optional) The viewport width (in pixels) to test against. Defaults to 800
tests.viewports.height Object/Number (optional) The viewport height (in pixels) to test against. Defaults to 600
tests.propertyValues Object A map of css properies and allowed values (or tokens)
tests.propertyValues.backgroundColor Array[string] (optional) An array of background-colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.borderTopColor Array[string] (optional) An array of border-top-colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.borderRightColor Array[string] (optional) An array of border-right-colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.borderBottomColor Array[string] (optional) An array of border-left-colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.borderLeftColor Array[string] (optional) An array of border-left-colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.color Array[string] (optional) An array of colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.marginTop Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.marginLeft Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.marginBottom Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.marginRight Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.outlineColor Array[string] (optional) An array of outline-colors that are allowed. Supported units: Hex, RGB, RGBA. Hex is converted to RGB prior to testing
tests.propertyValues.outlineStyle Array[string] (optional) An array of outline styles that are allowed. Supported values: 'dashed', 'solid' etc
tests.propertyValues.paddingTop Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.paddingLeft Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.paddingBottom Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.paddingRight Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.fontSize Array[string] (optional) An array of sizes that are allowed. Supported units: px
tests.propertyValues.fontFamily Array[string] (optional) An array of font-family declorations that are allowed
tests.propertyValues.fontWeight Array[string] (optional) An array of font-weights that are allowed. Supported values: '100', '200', '300'
